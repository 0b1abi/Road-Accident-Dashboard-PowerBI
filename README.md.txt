Road Accident Dashboard | Power BI

Project Overview
Developed an interactive Road Accident Analysis Dashboard in Power BI using 190K+ records. The dashboard analyzes accident trends, casualty severity, vehicle types, road and weather conditions, and monthly patterns to support data-driven decision-making.

Dashboard Preview

![Dashboard](screenshots/dashboard-overview.png)

 Tools Used
- Power BI
- Power Query
- DAX
- Excel

Dataset
- 190K+ Road Accident Records

Features
- Accident Trend Analysis
- Casualty Severity Analysis
- Vehicle Type Analysis
- Road Type Analysis
- Urban vs Rural Analysis
- Light Condition Analysis
- Interactive Filters

 Key Insights
- Total Casualties: 196K
- Total Accidents: 144K
- Cars recorded the highest casualties.
- Urban roads had more accidents than rural roads.
- Single carriageways showed the highest casualty count.
